Introduction

This is the posterdown_betterland template for the {posterdown} package! I was inspired by the twitter thread of Mike Morrison and wanted to apply the #betterposter concept to the reproducible (yet simple to use) functionality of the {posterdown} package (Thorne 2019). If you are not an R user don’t sweat as you do NOT need to use it at all! Feel free to use only the Markdown functionality of this package :)

Methods

  1. Install R and posterdown, see github
  2. Open the posterdown_betterland template
  3. Make all your poster dreams come true!
